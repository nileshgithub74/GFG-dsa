<h2><a href="https://www.geeksforgeeks.org/problems/sum-of-digit-is-pallindrome-or-not2751/1?page=2&difficulty=School&sortBy=submissions">Sum of Digit is Pallindrome or not</a></h2><h3>Difficulty Level : School</h3><hr><div class="problems_problem_content__Xm_eO" style="user-select: auto;"><p style="user-select: auto;"><span style="font-size: 18px; user-select: auto;">Given a number <strong style="user-select: auto;">n</strong>.Find if the digit sum(or sum of digits) of n is a Palindrome number or not.</span></p>
<p style="user-select: auto;"><span style="font-size: 18px; user-select: auto;">Note: A Palindrome number is a number that stays the same when reversed. Example- 121 , 131 , 7 etc.</span></p>
<p style="user-select: auto;"><strong style="user-select: auto;"><span style="font-size: 18px; user-select: auto;">Example 1:</span></strong></p>
<pre style="user-select: auto;"><span style="font-size: 18px; user-select: auto;"><strong style="user-select: auto;">Input:</strong>
n = 56
<strong style="user-select: auto;">Output: </strong>1
<strong style="user-select: auto;">Explanation:</strong>
The digit sum of 56 is 5+6=11.Since, 11 is a palindrome number.Thus, answer is 1.</span></pre>
<p style="user-select: auto;"><strong style="user-select: auto;"><span style="font-size: 18px; user-select: auto;">Example 2:</span></strong></p>
<pre style="user-select: auto;"><span style="font-size: 18px; user-select: auto;"><strong style="user-select: auto;">Input:</strong>
n = 98
<strong style="user-select: auto;">Output: </strong>0
<strong style="user-select: auto;">Explanation:</strong>
The digit sum of 98 is 9+8=17. Since 17 is not a palindrome,thus, answer is 0.</span></pre>
<p style="user-select: auto;"><span style="font-size: 18px; user-select: auto;"><strong style="user-select: auto;">Your Task:</strong><br style="user-select: auto;">You don't need to read input or print anything.Your Task is to complete the function <strong style="user-select: auto;">isDigitSumPalindrome()</strong> which takes a number n as input parameter and returns 1 if the Digit sum of n is a palindrome. Otherwise, it returns 0.</span></p>
